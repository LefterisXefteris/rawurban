<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Shopify product content -> description accordion | Remote product description text crosses from server-fetched data into interactive UI |
| product page client -> modal overlay state | User clicks open and dismiss a modal that traps focus over the page |
| navbar client -> internal suggestion navigation | Search overlay click targets navigate within the app |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-05 | I | `components/product/DescriptionAccordion.tsx` | mitigate | Render the description as plain JSX text only and keep the accordion as stateful show/hide logic; do not introduce `dangerouslySetInnerHTML` or HTML parsing. |
| T-02-06 | T | `components/product/SizeGuideModal.tsx` | mitigate | Use Radix dialog primitives for overlay/focus behavior instead of custom DOM mutation; keep all size-guide table content static and hardcoded for this phase. |
| T-02-07 | E | `components/navbar/SearchBar.tsx` | mitigate | Restrict overlay suggestions to hardcoded internal `Link` hrefs and avoid query-driven redirects or external destinations while full search remains deferred. |
| T-02-08 | D | `components/navbar/page.tsx` | accept | Search overlay toggling is purely local UI state with bounded animation and no network activity; low-risk client-side interaction only under ASVS L1. |
</threat_model>
